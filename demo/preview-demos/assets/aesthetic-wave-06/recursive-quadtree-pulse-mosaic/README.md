## Runtime derivation and evidence

2. p5.js crops source coordinates `(0, 50)` through `(960, 590)` and resizes that crop to a 96×54 buffer: 5,184 pixels / 20,736 RGBA bytes.
3. Every sampled pixel contributes luminance, RGB channels, terrain classification, and neighbour-difference texture. Integral images provide exact evidence summaries for every recursive node.
4. A node can split only when its sampled rectangle has sufficient real texture evidence. Human focus position and LOD controls adjust the inspection budget, but neither can fabricate detail in a flat region. Each internal node creates exactly four children.
5. The visible risk signal is calculated from the active leaf's dry-pixel share, water-pixel share, colour bias, and texture. It is not a prompt-derived or hard-coded map label.
