# Optical channel master

`optical-channel-master.jpg` is a fictional, same-origin calibration-film asset generated with the built-in ImageGen tool for the `moire-tunnel-zoom` demo.

It is functional evidence rather than decorative scenery. The live demo fetches the exact file, verifies its byte length and SHA-256, decodes it in the browser, downsamples a fixed crop to `96 × 54`, and samples those pixels. Measured edge density, directional gradients, luminance variation, saturation, and the bright aperture centroid determine the four inspection channels' ring counts, ellipse axes, colors, vanishing point, risk values, and conclusions.
