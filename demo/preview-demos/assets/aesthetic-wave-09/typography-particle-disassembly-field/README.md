## Runtime use

The title first exists as sampled pixels from an offscreen `LOW / TIDE` glyph mask. Each particle is then deterministically assigned to a high-information pixel candidate from this asset. A human changes the interpolation only through hover, captured mouse/touch/pen drag, keyboard, native range input, or visible buttons. There is no autoplay, preview-clock mutation, rehearsal, or synthetic input path.
