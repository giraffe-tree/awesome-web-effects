| Field | Allowed value | Meaning |
| --- | --- | --- |

| `State` | `next-pilot` | Explicitly selected as the next pilot; an ImageGen candidate may already exist. |
| `State` | `in-progress` | Implementation has started but has not passed its verification gate. No entry currently uses this state. |
| `State` | `verified` | Implementation and capture checks passed, but publication completion has not yet been declared. No entry currently uses this state. |
| `State` | `complete` | The current optimization wave explicitly marked the item complete. |
| `State` | `external-only` | The catalog item is represented by upstream official material and has no local runnable Demo in the audited scope. This is a scope constraint, not a licensing conclusion. |

| `Wave` | `W1 · complete` | A W1 pilot passed implementation, capture, publication, and push gates. |
