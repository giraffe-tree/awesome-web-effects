# Image generation prompts

The six AI-generated images are atmosphere-only backgrounds. Project UI, statistics,
Chinese copy, links, and GIF frames are composited from the repository by
`build_carousel.py` so that the final carousel stays accurate.

## 01 — Cover

```text
Use case: ads-marketing
Asset type: Xiaohongshu carousel slide background, portrait 3:4
Primary request: Create a premium abstract cover background inspired by a curated web interaction atlas.
Scene/backdrop: deep midnight indigo gradient with a subtle horizon glow, sparse micro-stars, a soft blush-pink radial light and one elegant translucent geometric portal in the lower-right.
Style/medium: editorial digital art, restrained glassmorphism, refined product-design campaign, high-end developer tool aesthetic.
Composition/framing: vertical 3:4, generous clean negative space across the upper-left and center-left for later typography; visual interest concentrated in lower-right; strong depth but not busy.
Lighting/mood: cinematic, confident, curious, luminous.
Color palette: #080d22, #161735, dusty pink #e9a0b4, warm ivory, muted periwinkle.
Constraints: background only; no words, no letters, no numbers, no logos, no UI screenshots, no watermark, no border, no people.
Avoid: generic neon cyberpunk, bright rainbow gradients, clutter, illegible pseudo-text, stock-photo look.
```
