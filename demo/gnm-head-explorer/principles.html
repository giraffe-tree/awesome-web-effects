THE COMPUTATION PIPELINE

把复杂的人脸变化，压缩成少量可解释参数。

关键不是生成更多多边形，而是让每种变化发生在正确的空间里，并保持顶点一一对应。

  1. A

    统一拓扑

    大量三维扫描先被配准到同一套顶点结构。于是第 8230 号顶点在每张脸上表达同一个局部位置，模型才有可能学习“如何变化”。

  2. B

    学习统计基底

    身份差异被压进 253 个方向，表情变化被压进 383 个方向。每个方向都是覆盖全部顶点的一组位移，而参数 β、ψ 决定叠加多少。

  3. C

    骨骼与蒙皮

    颈部、头部和两只眼球组成四关节层级。线性混合蒙皮按照每个顶点的权重传播旋转；姿态修正项 P(θ) 再补偿单纯蒙皮产生的形变误差。

  4. D
    βψθ→V

    同一函数，双向使用

    给定参数可以合成新头模；反过来，也可以优化参数去贴合图像、视频或扫描。因为计算过程可微，它能直接嵌入 PyTorch、JAX、TensorFlow 的训练流程。

READ THE MODEL CORRECTLY

GNM 是几何底座，
不是完整数字人。

✓它擅长

  • 稳定、拓扑一致的人头几何
  • 解耦身份、表情、视线与姿态
  • 合成数据、动画、拟合和神经渲染
  • 提供 NumPy、PyTorch、JAX、TensorFlow 实现

×它不直接提供

  • 头发、皮肤材质、毛孔和照片级着色
  • 从单张照片开箱即用地重建真人
  • 完整人体、服装和制作级 DCC 插件
  • 覆盖所有身份群体的无偏统计空间

数据边界：官方说明当前身份采样器使用二元性别与四个宽泛人口类别。这些标签无法完整表达真实世界的人类差异，在人脸分析与生成应用中需要单独评估代表性和公平性。
