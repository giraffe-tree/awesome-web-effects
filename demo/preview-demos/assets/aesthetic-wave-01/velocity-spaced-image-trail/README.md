# Velocity-spaced image trail assets

Four original, AI-generated editorial photographs for the `velocity-spaced-image-trail` demo. They form a fictional coastal field-study sequence: cohesive enough to read as one shoot, but intentionally distinct in subject, silhouette, and color so each frame remains recognizable in a fast image trail.

Generated on 2026-07-21 with the built-in OpenAI ImageGen tool (`photorealistic-natural`). The generated PNGs were mechanically converted to 86-quality JPEG and resized to 960 × 640 px with macOS `sips`; no compositing, retouching, or third-party source material was used. All locations, structures, and survey objects are fictional. The images do not document a real site and are not associated with any brand.

## Files and use

- `tidal-plate.jpg` — 960 × 640 px; cool reference frame and trail card 01.
- `dune-marker.jpg` — 960 × 640 px; warm reference frame and trail card 02.
- `signal-cloth.jpg` — 960 × 640 px; high-contrast geometry reference and trail card 03.
- `fog-pavilion.jpg` — 960 × 640 px; green architectural reference and trail card 04.

The demo cover-crops these images into a large inspection field and smaller p5/Canvas trail cards. JPEG compression and the 960 px long edge are deliberate: the images remain clear at the demo's supported 720 × 405 viewport while keeping the preview lightweight.
