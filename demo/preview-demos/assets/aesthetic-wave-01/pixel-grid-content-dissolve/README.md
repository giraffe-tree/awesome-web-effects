# Pixel Grid Content Dissolve — original image assets

Both images in this directory are original AI-generated visual inputs created specifically for the `pixel-grid-content-dissolve` demo. They depict fictional field-research sites and do not represent real facilities, brands, or locations.

## Disclosure

- Generation tool: OpenAI built-in ImageGen (`image_gen`)
- Generation date: 2026-07-21
- AI disclosure: both source photographs were generated with AI; no third-party photographic assets are included.
- Local processing: each generated PNG was resized to 960 × 640 and converted to JPEG at quality 88 with macOS `sips`.
- Intended use: full-bleed primary imagery in two distinct semantic dispatch content trees. The p5 canvas samples their pixels along the user-controlled replacement boundary.
