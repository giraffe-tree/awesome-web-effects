# Rope material board

- `rope-material-board.jpg` is an AI-generated fictional material-library photograph created for this demo with the built-in ImageGen tool on 2026-07-21.
- It depicts gold, navy, coral and neutral braided rope swatches on a charcoal work surface; there are no brands, people or claims of real-world origin.
- The browser fetches and decodes the committed JPEG, verifies its exact byte length and SHA-256, then samples a 96×64 raster. Those pixels determine the available rope materials, stroke width, coupling, load estimate, contrast and proof decision.
- The image is not used as an ornamental background. If it fails to load or its exact source hash changes, the demo fails readiness and the preview gate.
